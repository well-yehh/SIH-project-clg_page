Our Project

This website is designed as a Collage dashboard for students and faculty. It allows students to post doubts, view faculty replies, and see notices and timetables. Faculty can manage doubts, upload notices, and provide course schedules in an organized and professional manner.

Technologies Used

  • HTML5: Structured all web pages, forms, and content sections.
  • CSS3: Styled the pages with responsive layouts, colors, shadows, and smooth user interface elements.
  • JavaScript (ES6): Added interactivity such as posting doubts, replies, reactions, notices, timetable management, and storing data locally.
  • LocalStorage: Used for storing notices, doubts, reactions, and timetable data persistently on the browser.
  • Google Fonts: Applied clean typography with the Inter font for a modern look.

Features

  • Student and Faculty login with personalized interaction.
  • Student doubt board with subject selection and teacher replies.
  • Faculty notice board with optional photo attachments.
  • Timetable upload and management for different courses and sections.
  • Interactive reactions (like/dislike) and sorting by popularity.
  • Responsive design for desktop and mobile devices.

Our Goal

Our goal is to provide a smooth, user-friendly, and efficient platform for students and faculty to manage doubts, notices, and schedules digitally. This project demonstrates the use of modern web technologies while keeping the interface simple and professional.
